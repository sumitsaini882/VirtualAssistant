# 🤖 AI Voice Assistant

A powerful AI-based Voice Assistant inspired by Jarvis, built using modern web technologies.
This assistant listens to your voice, understands commands using AI, and performs real-time actions like searching the web, speaking responses, and more.

## ✨ Key Features

* 🎙️ **Voice Recognition** – Converts speech to text in real time
* 🧠 **AI-Powered Responses** – Uses Google Gemini API
* 🔊 **Text-to-Speech** – Speaks responses in Hindi/English
* 🌐 **Smart Actions**

  * Google Search
  * YouTube Search / Play
  * Open Apps (Instagram, Facebook, Calculator, Weather)
* 📜 **History Tracking** – Stores user queries
* 🔐 **Authentication System** – Login / Logout with JWT
* ⚙️ **Custom Assistant** – Personalize name & image
